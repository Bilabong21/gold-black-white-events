# BRC of SA Website

The BRC of SA Website is a modern web platform designed for the branches of the Bible Revelation Church of South Africa. The platform provides members, visitors, and church leadership with easy access to church information, events, announcements, sermons, ministries, and branch management tools.

## Overview

This system serves as the official digital platform for BRC of SA and its branches across South Africa. It aims to improve communication, administration, and member engagement through a centralized online presence.

## Features

### Public Features
- Church information and history
- Mission and Vision
- Upcoming events
- Latest announcements
- Sermon library
- Leadership profiles
- Ministries and church groups
- Branch locator
- Gallery
- Prayer request submission
- Contact forms
- Online donations

### Administrative Features
- Branch administrator authentication
- Announcement management
- Event management
- Sermon management
- Branch information management
- User role management
- Content publishing dashboard

## Church Structures Supported

- Synod
- NEC (National Executive Committee)
- PEC (Provincial Executive Committee)
- Youth Ministry
- Women's Ministry
- Men's Ministry
- Local Branches

## Technology Stack
